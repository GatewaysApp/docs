# Compare Gateways

Gateways is a **multi-cloud architecture console**: a visual canvas for real AWS, Azure, and GCP resources, workspaces, and APIs. These pages contrast that model with **hosted PaaS and edge platforms** that optimize for app deploys on their own infrastructure.

| | Gateways | Typical PaaS / edge |
| --- | --- | --- |
| **Primary job** | Design, connect, and operate cloud infra you own | Ship apps to vendor-managed runtime |
| **Cloud** | Your accounts (AWS, Azure, GCP) | Provider’s stack |
| **Model** | Resources, connections, environments | Services, regions, build pipelines |
